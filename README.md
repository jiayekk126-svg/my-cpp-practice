# C++ 学生成绩管理系统 (Student Management System)

简洁的练习项目，用于演示基于二进制文件的固定长度记录读写、按索引快速访问和简单的命令行交互。

## 功能
- 添加 / 删除 / 修改 / 查询 学生记录
- 按索引快速访问（直接通过文件偏移读取记录）
- 列出所有槽位（包含已删除槽位）
- 成绩统计（每门课的平均、最高、最低分）

## 主要文件
- `studentManagementSystem.cpp`：主程序源文件，包含程序逻辑与交互菜单。
- `students.dat`：程序运行时创建的二进制数据文件（自动创建）。
- `Project1.sln` 等：Visual Studio 工程文件（可选）。

## 数据格式说明
- 使用固定长度的 `struct Student`（不含 `std::string`），直接按二进制写入/读取，以便按记录槽位索引进行随机访问。
- 文件开头保存一个 `FileHeader`（记录槽位数），随后按固定大小存放每条 `Student` 记录。
- 注意：该二进制格式与编译器、结构体对齐和平台有关，跨平台或更改结构体字段时会影响兼容性。

## 构建要求
- 使用 C++14 编译。
- 在 Windows 上建议使用 Visual Studio 打开 `Project1.sln` 并编译。
- 使用 g++ 的命令示例：
  - `g++ -std=c++14 studentManagementSystem.cpp -o sms`

## 运行示例
- 通过命令行运行生成的可执行文件：
  - `./sms` （在 Windows 上双击或在 cmd/PowerShell 中运行 `sms.exe`）
- 程序首次运行会在当前目录创建 `students.dat`，保存记录。

## 使用说明（交互菜单）
1. 添加学生
2. 删除学生（按学号）
3. 修改学生（按学号）
4. 按学号查询
5. 按索引快速访问（从 0 开始）
6. 列出所有槽位（含已删除项）
7. 成绩统计
8. 退出

在添加/修改时，程序支持输入性别为 `男`/`女`（也兼容 `M`/`F`），内部以 `M`/`F` 存储并显示为中文。

## 已知注意事项
- 由于程序使用二进制写入 `struct`，请避免在 `Student` 中使用 `std::string`，避免改变字段顺序或类型以保持文件兼容性。
- 程序对中文编码的处理依赖于终端与操作系统编码，显示/输入中文姓名时可能需要合适的控制台编码（例如 Windows 的 GBK/UTF-8 设置）。
